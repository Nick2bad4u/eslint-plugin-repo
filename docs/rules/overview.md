# eslint-plugin-repo-compliance

This plugin enforces repository policy files and provider-specific automation conventions.

## What it checks

- Baseline repository hygiene (`README`, `LICENSE`, `SECURITY`, `CONTRIBUTING`, `SUPPORT`, `CODEOWNERS`, templates)
- GitHub compliance (Dependabot and workflow requirements)
- GitLab compliance (CI and template requirements)
- Bitbucket compliance (Pipelines configuration)
- Codeberg/Forgejo compliance (workflow requirements)
