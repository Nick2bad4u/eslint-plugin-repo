# Changelog

All notable changes to `eslint-plugin-repo-compliance` are documented in this file.

## 0.1.0 - 2026-04-23

### Added

- Initial scaffold for repository compliance linting.
- Provider-specific presets for GitHub, GitLab, Bitbucket, and Codeberg/Forgejo.
- Baseline repository policy rules (license, templates, security, support, workflows, and CI config checks).
